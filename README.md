## Getting Started

### Prerequisites

* Run TailwindCss
  ```sh
  npm run dev
  ```

1. Clone the repo
   ```sh
   git clone https://github.com/huthoinguyn/ecommerce_mvc_oop.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
