## Getting Started

### Prerequisites

Run tailwindcss
* npm
  ```sh
  npm run dev
  ```

### Installation


1. Get a free API Key at [https://example.com](https://example.com)
2. Clone the repo
   ```sh
   git clone https://github.com/huthoinguyn/ecommerce_mvc_oop.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
